# Netflix Pro Analyzer

A Python GUI application built with Tkinter that allows users to:
- Search movies using OMDb API
- Filter Netflix dataset by genre
- View movie details
- Add and store favorites permanently

#Features
- Live movie search
- Genre filtering (Netflix dataset)
- Favorites system with persistent storage
- Simple Netflix-like UI

##Tech Stack
- Python
- Tkinter
- Pandas
- Requests API

## Run Project
```bash
pip install -r requirements.txt
python app.py
